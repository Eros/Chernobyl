About Highlander

Highlander is an open source game all about exploration. Inspired by the game Minecraft the project originally started for college but quickly became something bigger that I (rapid) enjoyed developing over time.

The adventure ahead

Programming is an adventure. For me it's all about learning and advancing current skills to an entire new level. The goal with Highlander is to make it community based to allow any developer no matter what their skill set is to give them a chance to show and learn their current knowledge of programming.

Originally I wanted to make Highlander just for fun but I then saw what could be done with it. Having a community built around programming will give others the freedom to do what they want with the game (as long as its appropriate) and go wild with the world of code.
